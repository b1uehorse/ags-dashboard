# ags-dashboard

Свой мини-фреймворк поверх AGS v3 / Astal на Hyprland (CachyOS, Betrayer).

Цель — **конструктор** виджетов: новый тайл собирается из примитивов (`<Tile>`, `<CircularMeter>`, `<IconButton>` и т.д.) за минуты, а не верстается с нуля.

Первый пилот — повтор rxyhn-дашборда (`src/dashboards/rxyhn.tsx`). Дальше — свои экраны.

## Структура

```
src/
  primitives/   — переиспользуемые кирпичи
  widgets/      — композиции под конкретные роли
  services/     — обёртки над astal + свои источники
  dashboards/   — финальные композиции (окна)
  theme.ts      — единый источник цветов/размеров/шрифтов
  app.ts        — регистрирует все dashboards
docs/           — каталог примитивов, сервисов, тема, рецепты
snippets/       — copy-paste заготовки
```

## Запуск

```bash
cd ~/.config/ags && ags run
```

## Принципы

Подробности — в [CLAUDE.md](CLAUDE.md). Коротко:

1. Документация пишется синхронно с кодом.
2. Никаких цветов/размеров в компонентах — всё через `theme.ts`.
3. Примитивы — голые и переиспользуемые, виджеты — с бизнес-логикой.
4. Сервисы оборачиваются в свой `services/` слой, виджеты подписываются на него.
